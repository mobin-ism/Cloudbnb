**Airbnb Clone - Ruby on Rails Learning Project**

This project is a clone of the popular Airbnb platform, built with Ruby on Rails. The main goal is to learn and practice the fundamentals of Ruby on Rails while developing a fully functional application similar to Airbnb, allowing users to book and list properties.

### Features:
- User authentication (Sign up, Login, Logout)
- Property listing with details (price, location, images, etc.)
- Search and filter options based on location, price, etc.
- Booking functionality
- User profiles (hosts and guests)
- Review and rating system

### Technologies Used:
- Ruby on Rails
- PostgreSQL
- HTML, CSS, JavaScript (for front-end)
- Bootstrap for styling

### Objectives:
- Gain hands-on experience with Ruby on Rails
- Understand MVC architecture in Rails
- Learn about integrating databases and handling user interactions
